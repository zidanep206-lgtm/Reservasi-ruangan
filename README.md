# Sistem Reservasi Ruangan Kampus

## Deskripsi

Sistem Reservasi Ruangan Kampus adalah aplikasi berbasis PHP Native dan MySQL yang digunakan untuk mengelola peminjaman ruangan di lingkungan kampus.

## Fitur

- Login Multi Role
- Dashboard Statistik
- Manajemen User
- Manajemen Ruangan
- Reservasi Ruangan
- Approval Reservasi
- Kalender Reservasi
- Upload Surat PDF
- Detail Reservasi

## Teknologi

- PHP Native
- MySQL
- Bootstrap 5
- FullCalendar
- PDO
